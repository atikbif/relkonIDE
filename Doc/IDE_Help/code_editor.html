Редактор кода

Редактор кода представляет собой текстовый редактор с подсветкой синтаксиса языка Relkon и возможностью сворачивания/разворачивания отдельных блоков программы (процессов и блока объявления переменных).

Все строки программы пронумерованы. Это облегчает ориентирование по коду при отладке программы и поиске текста (более подробно поиск текста описан в разделе Меню).Содержимое каждого процесса программы может находиться в свёрнутом или развёрнутом состоянии. В свёрнутом состоянии не отображается тело процесса кроме заголовка(#PROCESS), а напротив номера строки заголовка отображается знак "+". Развернуть тело процесса можно щёлкнув мышкой по этому знаку. При этом знак поменяет своё состояние на "-". Повторный щелчок опять свернёт процесс. При открытии существующего проекта все процессы по умолчанию отображаются в свёрнутом виде.

В редакторе кода доступны следующие горячие клавиши:

  • Alt+Wheel (горизонтальная прокрутка текста)
  • Shift+Wheel (постраничная прокрутка текста)
  • Ctrl+C (если есть выделенный фрагмент он копируется в буфер обмена, если нет - в буфер копируется текущая строка)
  • Ctrl+V (вставка текста из буфера обмена)
  • Ctrl+X (вырезка выделенного фрагмента или текущей строки в буфер обмена)
  • Shift+Del (удаление текущей строки)
  • Shift+Home (выделение фрагмента текста от текущей позиции курсора до начала строки)
  • Shift+End (выделение фрагмента текста от текущей позиции до конца строки)
  • Shift+Tab (сдвиг строк, содержащих выделение, влево)
  • Tab (сдвиг строк, содержащих выделение, вправо)
  • Insert (включение/выключение режима замены символов)
  • Ctrl+Z (отмена последней операци)
  • Ctrl+Y (повтор отменённой операции)
