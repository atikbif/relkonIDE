Обзор возможностей

Среда программирования RIDE предназначена для разработки и отладки программного обеспечения, управляющего работой программируемых логических контроллеров производства ООО "Контэл".

Функционально среду можно разбить на несколько блоков:

  • Редактор кода
  • Отладчик
  • Редактор пульта
  • Настройки контроллера

    • Редактор кода представляет собой текстовый редактор в котором на языке Relkon (адаптированная версия языка Си) описывается алгоритм работы программы.

    Отладчик служит для отладки и тестирования работы контроллера.

    Редактор пульта формирует сообщения на символьном дисплее. В зависимости от типа контроллера пульт может быть встроенным или выносным.

    Каждый контроллер имеет блок энергонезависимой памяти. В ней хранятся системные настройки (такие как сетевой адрес, тип протокола и скорость порта для интерфейсов контроллера) и заводские установки для конфигурации режимов работы автоматики.
