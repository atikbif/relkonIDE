Чтение программы и настроек из контроллера

Чтение программы возможно только с помощью утилиты Загрузчик, подробная работа с которой описана в разделе справки "Утилиты". Непосредственно же из среды возможно только чтение настроек. Для этого необходимо подключить контроллер к компьютеру через адаптер USB/RS485 и выбрать пункт меню "Проект->Прочитать настройки" или нажать F9. Можно перейти и в окно настроек и нажать там кнопку "прочитать". При этом среда начнёт поиск контроллера и при его обнаружении считает все его настройки (настройки интерфейсов и заводские установки), которые отобразятся в соответствующих полях. Для сохранения настроек необходимо либо сохранить либо перекомпилировать проект.
